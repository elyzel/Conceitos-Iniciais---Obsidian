# 🚀 Meu Cofre de Formatações & Aprendizados

Bem-vindo(a) ao meu repositório de estudos! O objetivo deste espaço é centralizar, organizar e documentar todas as minhas anotações sobre **instruções de formatação, Markdown e sintaxe de organização**.

---

## 🛠️ A Ferramenta: Obsidian

Todo o conteúdo deste repositório foi construído e pensado para ser utilizado no **Obsidian**, um dos softwares de notas mais poderosos da atualidade. 

O Obsidian transforma simples arquivos de texto em uma base de conhecimento interconectada (como uma "segunda mente"), permitindo que eu visualize a relação entre diferentes conceitos através do seu incrível **Graph View** e utilize recursos avançados como fórmulas matemáticas matemáticas e blocos customizados.

---

## 📂 O que você vai encontrar por aqui?

Neste repositório, os arquivos já estão estruturados para cobrir desde o básico até o avançado:
* **Guia de Markdown:** Como aplicar negrito, itálico, destaques e códigos de forma nativa[cite: 1].
* **Matemática Dinâmica:** Instruções exatas de como centralizar e renderizar fórmulas complexas com LaTeX[cite: 1].
* **Operadores Lógicos e Símbolos:** Tabelas rápidas de consulta para estruturar raciocínios lógicos.
* **Gestão de Conhecimento:** Estruturas focadas em links internos (`[[]]`) e manipulação visual do gráfico de notas.
